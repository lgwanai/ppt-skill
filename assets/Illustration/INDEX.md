## 使用方法

### 1. 直接引用 SVG 文件

```python
from pptx import Presentation
from pptx.util import Inches, Pt

prs = Presentation()
slide = prs.slides.add_slide(prs.slide_layouts[6])

# 添加插图
slide.shapes.add_picture(
    "assets/Illustration/undraw_ai-chat_ljb9.svg",
    Inches(1), Inches(1), Inches(4)
)
```

### 2. 修改颜色

编辑 SVG 文件中的 `fill` 属性：

```xml
<!-- 原始 -->
<path fill="#f9a826" d="..." />

<!-- 修改为主色调 -->
<path fill="#0070C0" d="..." />
```

### 3. 在 CLI 中搜索

```bash
# 搜索 AI 相关插图
ls assets/Illustration/ | grep ai-

# 搜索数据相关插图
ls assets/Illustration/ | grep data
```
